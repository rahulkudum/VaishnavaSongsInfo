Vaishnava Songs App Privacy Policy

This application does not collect any user data. Additionally, there is no database or backend connected to this application. All user data, such as bookmarks and preferences, is stored locally on the device. Users have the option to backup their data to Google Drive, where it is securely stored within a designated folder exclusive to the application (unreachable by the user). It is important to note that the application does not have the capability to access or alter personal data of the users stored in Google Drive.
